Skechers GO RUN Ride 11

Engineered for peak performance and all-day comfort, the Skechers GO RUN Ride 11 is a next-level running shoe designed for serious runners and comfort seekers alike. Featuring a lightweight, breathable upper and advanced carbon-infused cushioning, this running powerhouse delivers responsive energy return, superior traction, and a smooth ride on every stride.

Key Features:

  • ✔ Patented Skechers Arch Fit® insole system with podiatrist-certified arch support
  • ✔ Designed with 20 years of data and 120,000 unweighted foot scans for optimal shape and fit
  • ✔ Removable insole molds to your foot to reduce shock and evenly distribute weight
  • ✔ HYPER BURST ICE™: Dual-density cushioning for both softness and stability
  • ✔ HYPER ARC® technology enhances stride efficiency for a smoother transition
  • ✔ Carbon fiber-infused plate for added propulsion and spring in every step

Design Details:

  • ✓ Goodyear® Performance Outsole for excellent traction, durability, and stability
  • ✓ Lightweight, breathable engineered mesh and synthetic upper
  • ✓ Road-running performance lace-up design
  • ✓ Weight: 9.9 oz / 281 g (Men’s size 9)
  • ✓ Stack Height: 32mm (forefoot) – 38mm (heel) | Offset: 6mm
  • ✓ Machine washable for easy care
  • ✓ Subtle Skechers GO RUN® logo detailing

Whether you’re pushing the pace or going the distance, the Skechers GO RUN Ride 11 delivers the responsive cushioning and technical features you need for high-performance running — with comfort that lasts beyond the finish line.
